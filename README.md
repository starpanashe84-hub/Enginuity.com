# Enginuity — Where Engineering Meets Infinity

**Enginuity** is a premium, cinematic learning platform designed specifically for engineering students. It simplifies complex theoretical concepts through high-fidelity visualisations, interactive problem sets, and an AI-powered study guide.

## 🪐 Features
- **Cinematic Experience**: A minimalist, high-contrast dark theme with cosmic aesthetics.
- **Interactive Disciplines**: Over 15 engineering branches covered, from Aerospace to Nuclear.
- **AI-Powered Guide**: (Coming Soon) Personalised AI tutor for technical queries.
- **Vibrant Community**: Collaborate with thousands of engineers globally.

## 🛠️ Tech Stack
- **Structure**: Vanilla HTML5
- **Styling**: Modern CSS3 (Variables, Grid, Flexbox)
- **Logic**: Vanilla JavaScript
- **Typography**: Space Grotesk, Inter, Roboto Mono

## 🚀 Getting Started
To run this project locally:
1.  Clone the repository: `git clone <your-repo-url>`
2.  Open `index.html` in your browser or use a live server.
